A responsive travel landing page built with HTML and CSS, featuring a navigation bar, search box, hero section, and call-to-action buttons.
This project is perfect for practicing layout, typography, and interactive UI components.
Features
Responsive navigation menu with hover effects
Search bar with icon and rounded styling
Hero section with background image overlay
Centered title and call-to-action buttons
Font Awesome icons integration
Fully built with HTML and CSS, no JavaScript
Built With
HTML5
CSS3
Font Awesome
Purpose
Created to practice:
Responsive design and layout
Hero section styling with absolute positioning and transforms
Navigation menu and hover effects
Form elements styling (search box)
Integration of icons using Font Awesome
